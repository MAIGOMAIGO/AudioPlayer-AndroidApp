# AudioPlayer-AndroidApp  
Android Studioを使って音楽再生アプリを作りました。  

## 開発環境  
Android Studio Arctic Fox | 2020.3.1  
対象のAndroid：23～32  
試したAndroid：
  SO-01L  
  Android version 10  
  API Level 29
